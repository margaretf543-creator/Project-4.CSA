# Project-4.CSA
# Car Sales Advertisements Dashboard

## Project Description
This is an interactive web application that explores car sales advertisement data in the US. The tool allows users to analyze vehicle pricing, condition, and other characteristics through interactive visualizations.

## Methods and Libraries Used
- **Streamlit** — web application framework
- **Pandas** — data manipulation and analysis
- **Plotly Express** — interactive data visualizations

## Dataset
The dataset contains 51,525 car sales advertisements with information about price, model year, condition, odometer reading, fuel type.

## Features
- Distribution of car prices histogram
- Number of cars by condition histogram
- Price vs Model Year scatter plot
- Filter to show only cars under $50,000
